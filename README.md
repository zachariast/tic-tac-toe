## Tic Tac Toe Project

A simple tic tac toe project build with React (Redux, typescript, hooks)


## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

node version 14.17

`npm i`  

To Start Server:

`npm start`  

To visit app:

`localhost:3000`
